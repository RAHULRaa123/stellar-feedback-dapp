# Stellar Connect Wallet

## Description
This is a simple Stellar dApp built on testnet using Freighter wallet.  
It allows users to connect their wallet, view XLM balance, and send transactions.

## Features
- Connect Freighter wallet
- Disconnect wallet
- Display XLM balance
- Send XLM to another address
- Show transaction success or failure

## Tech Stack
- React.js
- Stellar SDK
- Freighter Wallet

## Setup Instructions
1. Clone the repository  
2. Run npm install  
3. Run npm start  
4. Open http://localhost:3000
